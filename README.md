# Lectura

Este proyecto hecho en Node.js consta de poder cargar archivos '.txt' para luego poder leerlo de una forma más cómoda.
Se puede personalizar tanto estilos de la fuente como la velocidad de lectura.

### Instalar librerías

```
npm install
```

### Ejecutar servidor

```
npm run start
```

### Compilar aplicación

```
npm run build
```
